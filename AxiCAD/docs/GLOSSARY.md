# Glossary

> Single source of truth for AxiCAD terminology.
> All specs reference this file instead of re-defining terms.

---

| Term | Definition |
|------|------------|
| **Workspace** | Top-level container for an entire neural network layout project |
| **Layer** | A discrete vertical level within the workspace |
| **Department** | A logical grouping of shards within a layer |
| **Shard** | A volumetric region containing somas (neurons) |
| **Socket** | A connection point on a shard boundary for inter-shard communication |
| **Tract** | A routed bundle of axonal connections between sockets |
| **Voxel** | The smallest discrete unit of the 3D grid |
| **Grid** | The discrete 3D coordinate system underlying all spatial calculations |
| **Soma** | A single neuron unit occupying one or more voxels within a shard |
| **TOML** | The declarative file format used for serializing network layouts |
| **ADR** | Architecture Decision Record — a document capturing a non-obvious design choice |
