# AxiEngine project skills

**Canonical skill sources live here** (`axi-SKILLS/`).  
Grok/TUI also needs them under `.grok/skills/<name>/` — keep those as **copies or junctions** of the folders below (not a second conflicting invent).

| Skill | Use |
|---|---|
| `axi-research` | Research / LP / microcircuit numbers: archive → unknown → bio; no invented knobs |

## For agents

- Prefer reading `axi-SKILLS/<name>/SKILL.md`.
- Also: `GEMINI.md` §13.0, `docs/engine/research/RULES.md` §0, `artifacts/agent-tasks/DECISIONS_RU.md`.

## Add a skill

1. Write `axi-SKILLS/<name>/SKILL.md` (frontmatter `name` + `description`).
2. Copy/junction to `.grok/skills/<name>/` so slash menu / auto-invoke works.
