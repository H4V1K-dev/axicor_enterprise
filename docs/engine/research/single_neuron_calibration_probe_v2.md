# Адаптивная калибровка GLIF-нейрона (V2 Adaptive Calibration Probe)
*(single-neuron-physics-calibration-probe-v2-adaptive)*

Этот отчет содержит результаты исследования влияния существующих адаптивных механизмов AxiEngine (гомеостатический сдвиг порога, адаптивная утечка, послеспайковый сброс AHP) на точность воспроизведения свойств референсных биологических нейронов. Симуляции проводились по 5 режимам на оптимизированной сетке параметров.

## 2. Анализ влияния адаптивных параметров

### 1. Улучшение для Scnn1a (возбуждающий нейрон L4)
- Включение послеспайкового сброса потенциала действия (`glif_ahp_reset` с амплитудой $A_{\text{ahp}} = 10$ mV) снизило среднеквадратичную ошибку f-I кривой с **5.85** до **5.32**.
- Дополнительный сброс потенциала ниже уровня $V_{\text{rest}}$ (до -83 mV при биологическом покое -73 mV) имитирует затяжную следовую гиперполяризацию (AHP), сглаживая частоту спайков на высоких токах стимуляции.

### 2. Почему PV и SST остались на Baseline
* **PV (интернейроны Pvalb)**:
  - Pvalb-клетки биологически обладают сверхбыстрой регулярной импульсацией без выраженной адаптации. Целевой индекс адаптации для них был задан как `0.0`, поэтому базовый LIF-модель идеально описывает их динамику.
* **SST (интернейроны Somatostatin) и Cux2**:
  - Данные клетки обладают выраженной биологической адаптацией. Однако оптимизатор предпочел `lif_baseline`.
  - **Причина**: Сильное взаимное влияние (`coupling`) параметров. При активации `homeostasis_penalty` или `adaptive_leak_gain` порог генерации спайков растет с каждым импульсом, из-за чего общая частота разрядов резко снижается.
  - Поскольку масштаб тока (`current_scale = 0.02`) был зафиксирован под ненасыщаемый Baseline, включение адаптации привело к резкому недовозбуждению клеток (спайки прекращались), что вызывало колоссальный рост RMSE. Для калибровки адаптации требуется *совместный* поиск масштаба тока и параметров гомеостаза.

## 3. Оценка достаточности физики AxiEngine

* **Вывод**: Существующая физика GLIF в AxiEngine (описанная в `physics::update_glif_voltage` и `physics::is_glif_spike`) достаточна для первого грубого f-I зондирования. Тем не менее, **полная достаточность/адекватность адаптивных механизмов не доказана**, поскольку в 3 из 4 случаев наилучшим результатом по комбинированной оценке остался простой базовый режим `lif_baseline`, а для Scnn1a (`314900022`) было получено лишь небольшое улучшение через сброс потенциала (`glif_ahp_reset` c уменьшением RMSE с 5.85 до 5.32).

### Фактические результаты оптимизатора:
- `313861608` (PV): best = `lif_baseline`
- `490376252` (Cux2): best = `lif_baseline`
- `314900022` (Scnn1a): best = `glif_ahp_reset` (небольшое улучшение)
- `324493977` (SST): best = `lif_baseline`

### Ключевые ограничения текущего зондирования (Probe V2):
1. **Отсутствие совместной калибровки**: Параметры `current_scale`, `leak_shift` и `refractory` были жестко зафиксированы около наилучших значений Baseline V1. Это исключило возможность компенсации роста порогов при активации гомеостаза за счет пропорционального увеличения силы тока.
2. **Эвристическая адаптация**: Целевые показатели адаптации (`target_adaptation`) были заданы эвристически на основе качественного поведения клеток, а не извлечены напрямую из экспериментальных ISI NWB-файлов.
3. **Приоритет f-I формы**: Значение `sim_adaptation = 0.0` у трех выбранных бест-конфигураций наглядно демонстрирует, что текущая loss-функция предпочла минимизировать ошибку формы f-I кривой ценой полного отказа от адаптации.

## 4. Следующие шаги

1. **Production-код физики AxiEngine в данный момент не меняется**. Математические ядра остаются в текущем состоянии.
2. **Следующий шаг исследования**: Извлечь реальные интервалы спайков (ISI) из NWB-файлов на уровне высокоинтенсивных свипов (high-current sweeps) для точного количественного расчета биологической адаптации.
3. **Двумерная калибровка**: Только после этого провести полноценный совместный поиск по сетке: `current_scale + homeostasis_penalty + adaptive_leak_gain`.
