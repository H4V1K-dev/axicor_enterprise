# Night Phase Activity-Aware Pruning (v0.6)

This research experiment evaluates trace and coactivity-aware synapse pruning against the traditional absolute weight floor pruning policy, under identical budget-matched pruning pressure.

## Objective
Evaluate if utilizing activity-aware counters (`coactivity_hits`, `pre_hits`) and dual structural traces (`short_trace`, `long_trace`) in a deterministic night pruning score allows the network to select and prune background noise synapses while protecting weak, rare, but functionally relevant (stimulated) pathways.

## Pruning Policies
1. **`passive_recovery_only`**: Baseline control policy with no pruning.
2. **`absolute_weight_floor_1498`**: Traditional pruning policy where synapses with weight magnitude below $1498 \times 2^{16}$ are pruned.
3. **`trace_score_global_budget`**: Score-based pruning using a deterministic `prune_score`, budget-matched to prune the same *total* number of synapses as the absolute floor policy.
4. **`trace_score_projection_budget`**: Score-based pruning budget-matched *per projection class*, ensuring that the projection composition is perfectly aligned with the absolute floor policy.

## scoring rule
The deterministic score used for trace-aware policies is:
$$\text{prune\_score} = \text{low\_weight} + 2 \times \text{low\_coactivity} + 2 \times \text{low\_trace} + \text{negative\_trend} - \text{protection\_bonus}$$

Where:
- **`low_weight`**: Weaker weights receive higher scores.
- **`low_coactivity`**: Lower coactivity ratio (coactivity hits / pre hits) receives higher scores.
- **`low_trace`**: Lower `long_trace` values receive higher scores.
- **`negative_trend`**: Synapses with an active LTD trend receive higher scores.
- **`protection_bonus`**: Protection bonuses are awarded to protect:
  - Synapses with high long trace values ($\ge 20$).
  - Synapses with high coactivity ratio ($\ge 40\%$).
  - Rare-but-useful pathways (low presynaptic rate but high coactivity ratio).
  - Target somas with critical diversity constraints (to avoid pruning a target's last useful connections).
